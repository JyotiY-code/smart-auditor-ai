# Smart Auditor AI

An AI-based document auditing system built with Python.

## Features

- PDF and TXT document extraction
- AI-style anomaly detection
- Risk scoring engine
- Structured audit report generation
- Automatic report saving

## Tech Stack

- Python
- PyPDF2
- Git & GitHub

## Current Status

🚧 Project under active development.

## Future Improvements

- Gemini AI integration
- OCR support
- Dashboard UI
- Multi-file auditing
- PDF report export

## How to Run

Install dependency:

```bash
pip install PyPDF2
```

Run project:

```bash
python main.py
```
